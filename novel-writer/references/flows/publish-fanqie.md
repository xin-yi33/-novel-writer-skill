# 发布到番茄小说（在线模式）

> 对应 SKILL.md B 节。将本地已生成的章节自动上传到番茄小说平台（fanqienovel.com）。

---

## B1. 检查浏览器 MCP 可用性

检查 `mcp__browser__*` 工具是否可用。如果不可用，告知作者无法使用在线模式，建议切换为本地模式。

## B2. 打开番茄小说作家后台

使用 MCP 浏览器工具打开 `https://fanqienovel.com/main/writer/book-manage`。

**关键点：**
- 使用 `mcp__browser__open` 工具，设置 `newTab=true`
- 等待页面加载完成

## B3. 处理登录

检查页面是否显示登录界面：
- 如果已登录，继续下一步
- 如果提示登录，**告知作者需要在浏览器中手动完成登录**，等待作者确认登录完成后再继续

## B4. 确认作品

1. 使用 `mcp__browser__snapshot` 或 `mcp__browser__screenshot` 查看页面内容，获取作品列表
2. 读取本地的 `novel/chapters/` 或 `novel/current_state.md`，获取小说书名
3. 询问作者：**"我看到您后台有以下作品：[列出作品]。请问我们要发布的是『[本地书名]』吗？"**
4. 如果作者确认，点击对应作品进入管理页面
5. 如果作者否认，询问正确的书名，然后从作品列表中找到并点击

## B5. 进入章节管理

1. 进入作品管理页后，查找并点击"章节管理"或"管理章节"按钮
2. 确认章节管理页面已加载

## B6. 上传章节

**单章上传：**
1. 点击"新建章节"或"添加章节"按钮
2. 在标题栏填入章节标题
3. 在内容区填入章节正文
4. **点击"存草稿"按钮**（不要点击发布/提交审核）

**多章上传（一次性生成多章时）：**
1. 对每一章重复执行：
   a. 点击"新建章节"
   b. 填写标题和内容
   c. 点击"存草稿"
2. 等所有章节都处理完后再通知作者

**注意事项：**
- 番茄小说编辑器的选择器可能比较复杂，使用 `mcp__browser__snapshot` 查看页面结构来找到正确的元素
- 如果找不到"新建章节"按钮，截图查看页面布局，尝试点击明显的管理/内容管理入口
- 内容粘贴时注意保持格式，分段要清晰
- 如果编辑器有字数限制提示，注意检查
- 存草稿后确认是否保存成功（检查页面提示或截图确认）

## B7. 完成

告知作者哪些章节已成功上传为草稿。
