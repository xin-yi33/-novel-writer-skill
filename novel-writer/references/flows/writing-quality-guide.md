# 写作质量参考

> 对应 SKILL.md J 节。为所有生成流程（写新章节、续写、卡文助手、长篇写作模式）提供写作质量标准。**生成章节时必须主动运用本节相关技巧并进行质量自检。**

---

## J1. 章节结构指南

### 标准章节结构

| 段落 | 占比 | 作用 | 必须包含 |
|------|------|------|---------|
| **开头钩子** | 前 20% | 抓住读者，制造继续阅读的欲望 | 即时冲突/危机、重大事件、情感冲击、行动场景 |
| **发展推进** | 中间 50-60% | 推进剧情或深化人物 | 新信息揭示、关系变化、问题升级、角色成长 |
| **高潮时刻** | 后 15-20% | 本章情感或动作最高点 | 动作高潮、情感高潮、心理高潮 |
| **结尾钩子** | 最后 5-10% | 留下悬念，让读者想看下一章 | 未解答的疑问、突转、新危机浮现 |

### 十种开头技巧

生成每章开头时，至少使用以下一种技巧：

1. **行动中开场 (In Media Res)** — 直接从冲突或动作的高潮点开始，不铺垫
2. **反常情境** — 呈现不符合常理的场景，激发好奇
3. **震撼对话** — 用一句惊人的对话开场，立即制造情感冲击
4. **倒计时开场** — 从时间压力开始，制造紧迫感
5. **重大发现** — 从发现关键线索或真相开始，迫使读者重新理解
6. **危机时刻** — 从角色面临最大危机开始，让读者关心角色命运
7. **谜团浮现** — 从无法解释的现象开始，让读者想解开谜题
8. **背叛开场** — 从信任崩塌开始，角色瞬间陷入绝境
9. **重大选择** — 从角色的艰难决定开始，让读者代入思考
10. **结局预告** — 从未来的关键时刻开始，制造"这一切是如何发生的"的好奇

**开头致命错误（绝对避免）：** 天气描写、日常流程（起床刷牙吃饭）、回顾上一章、缓慢铺垫（城市背景介绍）、平淡对话（你好吗我很好）、过度解释（讲述而非展示）。

### 章节类型与字数

| 章节类型 | 字数 | 核心事件 | 适用场景 |
|---------|------|---------|---------|
| 短章节 | 2000-3000 | 1 个主要事件 | 过渡衔接、单一场景聚焦 |
| 标准章节 | 3000-5000 | 2-3 个主要事件 | 大多数章节 |
| 长章节 | 5000-7000 | 3-4 个主要事件 | 高潮、大战、多线收束 |

### 章节类型分类

- **情节推进章** — 推动主线剧情，有明确事件进展和信息揭示
- **人物深化章** — 揭示人物背景、动机、内心冲突（必须与主线相关）
- **氛围营造章** — 建立特定情绪或紧张感（注重感官描写和节奏控制）
- **过渡衔接章** — 连接两个重大事件（保持简洁，但仍需一个不可删除的小事件）

## J2. 对话写作指南

### 核心原则

每句对话必须至少完成以下之一：
- **推动情节** — "我找到凶器了，在河边的草丛里。"
- **揭示人物** — "我不信任警察，他们从来不帮我这样的人。"
- **制造冲突** — "你骗了我。你从头到尾都在骗我。"
- **传达信息** — "炸弹将在三点引爆。"
- **表达情感** — "我...我不知道该说什么。"
- **制造悬念** — "你知道那天晚上真正发生了什么吗？"

**简洁原则：** 人们说话不写论文。"我想告诉你的是，我认为我们应该立刻离开这里。" → "我们得马上走。"

**真实原则：** 真实对话充满打断、迟疑、话题转移、话没说完、暗示而非明说。

### 潜台词 (Subtext)

好的对话，真正含义在表面之下：

| 直接（乏味） | 潜台词（有趣） |
|-------------|---------------|
| "我很生气。" | "没事。我挺好的。真的。" |
| "我喜欢你。" | "你今天看起来...不错。" |

**潜台词技巧：**
1. **话题转移** — 不正面回答，转向无关话题
2. **反问回避** — 用反问代替回答
3. **以动作代答** — 沉默和动作比语言更有力
4. **谈论他物** — 借其他事物表达真实想法

### 对话与动作结合

- **一致增强：** "我爱你。"她紧紧抱住他，眼泪流下来。
- **矛盾揭示真相：** "我完全支持你。"他目光看向别处，手在口袋里握紧拳头。
- **动作替代标签：** "你在撒谎。"她拍案而起。

### 对话节奏

| 节奏 | 特征 | 适用场景 |
|------|------|---------|
| **快节奏** | 短句、无标签、打断、单句成段不超过15字 | 争吵、追逐、紧急决策 |
| **慢节奏** | 长句、停顿、迂回、插入动作和沉默 | 告白、试探、审讯、告别 |
| **沉默** | 以停顿和动作代替语言 | 拒绝回答、震惊、情感崩溃 |

### 对话场景类型

- **争吵** — 短句、打断、重复强调、情绪升级
- **告白** — 迟疑、停顿、寻找词语、真诚或尴尬
- **暧昧** — 双关语、试探、暗示、话里有话
- **告别** — 未尽之言、克制、表面平静下的暗涌

### 对话常见问题（避免）

1. **信息倾倒** — 角色互相说已知信息（"正如你所知，公司成立于1995年..."）
2. **所有人说话一样** — 无法区分角色。解决：每个角色独特的说话方式
3. **过度礼貌** — 真实对话更粗糙（"帮我个忙"而非"能否请你..."）
4. **无意义的闲聊** — 除非有特殊目的
5. **说教** — 角色发表长篇哲学论述

## J3. 标题创作指南

### 题材-风格映射

| 题材 | 风格特征 | 常用结构 |
|------|----------|----------|
| 悬疑/推理 | 悬念感 | "谜、局、消失的X、第N个X" |
| 言情/甜宠 | 诗意/温暖 | 意象、四字短语、对比结构 |
| 奇幻/仙侠 | 宏大/古风 | "录、传、纪、诀、歌" |
| 科幻 | 未来感/哲思 | 技术意象、悖论结构 |
| 历史 | 厚重感 | 年代、地名、事件锚点 |
| 都市/现实 | 生活化 | 口语化表达、反差结构 |

> 跨题材时优先匹配核心冲突对应的风格，同时融入次要题材元素。

### 五种创作技巧

生成候选标题时，每个候选使用不同的技巧：

1. **核心冲突提炼法** — 从核心冲突提炼关键词（悬疑追查失踪案 → 《失踪者线索》）
2. **主角命名法** — 以主角特征/身份构建（深夜食堂老板 → 《深夜掌勺人》）
3. **意象隐喻法** — 核心意象暗示主题（真相与谎言 → 《雾中灯塔》）
4. **反差法** — 矛盾元素并置引发好奇（平凡与非凡 → 《小人物的大日子》）
5. **悬念留白法** — 不完全揭示，留下想象空间（《第三封信》《门后的世界》）

### 质量标准

- **字数：** 2-8 字为主流
- **必须避免：** "XX之道""XX的觉醒""XX纪元""XX崛起"等 AI 套路化结构；过于笼统的单字/双字标题；直接剧透的标题

## J4. 中文文学技法

### 白描

用最少的笔墨精确捕捉本质。名词和动词为主，形容词能砍就砍。

| 堆砌修饰（避免） | 白描（使用） |
|----------------|-------------|
| 璀璨的星空下，瑰丽的晚霞映照着静谧的湖面 | 湖面映着天光，水波不兴 |
| 他很愤怒 | 他握紧拳头，指节发白 |
| 她很伤心 | 她低头搅咖啡，勺子停了很久 |

**操作：** 写完一段后删掉所有形容词再读一遍，如果意思没变，那些形容词就是多余的。

### 留白

不说的部分比说的部分更有力量。
- **对话场景：** 在沉默的那一刻收尾，不写出回答
- **情感场景：** 写动作不写感受。"她把信折好放回信封"比"她心如刀绞"有力
- **转折场景：** 写到转折发生的那一瞬间停止
- **高潮场景：** 最高潮处戛然而止，不写角色的感想和总结

### 意象营造

用一个反复出现的具象物品承载抽象情感。
- 前期自然出现，不刻意强调
- 中期变化重复（花从新鲜到枯萎，钟从走动到停摆）
- 关键时刻集中爆发
- 一本书里意象不宜超过三个

### 草蛇灰线

在看似随意的细节中埋入线索。读者第一次读不会注意，重读时拍案。
- 伪装成无关紧要的闲笔
- 揭晓时不张扬，让读者自己发现联系
- 间隔至少五章以上
- 伏笔必须记录，防止遗忘

## J5. 节奏控制指南

### 句子节奏

- 连续三句长度相同就必须打破
- 动作段落：句子控制在 10 字以内
- 转折时刻：用一个极短句制造停顿（"他愣住了。""没有。"）
- 列举或铺排时才允许连续短句，其他时候长短交错

### 段落呼吸

段落是读者的呼吸节奏：
- **紧张场景：** 一段一两句话，甚至一句话一段
- **缓和场景：** 一段五六句话，允许展开
- **转折爆发：** 长段落中间突然插入单句段落，像一记鼓点

### 信息密度波浪

信息密度 = 单位字数里传达的新信息量。
- 高密度(动作/对话/揭示) → 低密度(沉淀/描写/内心) → 高密度(新冲突/转折) → 低密度(情感消化/环境过渡)
- 连续三段以上高密度 = 读者疲劳跳读
- 连续三段以上低密度 = 读者无聊弃书
- 每章应有 2-3 个密度波浪

## J6. 打破读者预期

可预测性是小说最大的敌人。使用以下技术：

1. **预期反转** — 建立对 A 走向的预期，转向 B。A 的铺垫必须足够真实，反转不能凭空出现。
2. **信息差利用** — 让读者知道角色不知道的事（制造焦虑），或角色知道读者不知道的事（制造悬念）。
3. **复杂动机** — 没有纯粹的善恶。当读者理解反派的立场，或发现主角不完全是正义的，故事立刻有深度。
4. **节外生枝** — 在看似已解决的问题上引入新变量。读者刚松口气，发现事情远未结束。

## J7. 写作质量检查清单

生成每章后逐项自查。**任何一项不合格就修改。**

### 开头检查
- [ ] 前 20% 是否极其吸引人？（如果不是，重写）
- [ ] 是否在第一段就建立冲突或紧张？
- [ ] 是否避免了所有致命错误？（天气、日常、回顾等）
- [ ] 读者是否会迫切想知道接下来发生什么？

### 内容检查
- [ ] 本章是否推进了主线剧情或深化了人物？
- [ ] 是否有核心事件（删除它会影响理解）？
- [ ] 是否展示而非讲述了关键信息？
- [ ] 结尾是否留下悬念钩子？

### 人物检查
- [ ] 人物行为符合已建立的性格？
- [ ] 对话能区分不同角色？
- [ ] 通过行动和对话表现性格，而非直接陈述？

### 对话检查
- [ ] 每句对话有目的（推进情节、揭示人物、制造冲突）
- [ ] 没有无意义对话，对话简洁自然
- [ ] 有潜台词，不是所有话都直说
- [ ] 对话标签使用得当，不过度使用副词

### 节奏检查
- [ ] 句子长度有变化，没有连续三句长度相同
- [ ] 段落长度有变化，紧张场景用短段落
- [ ] 信息密度有高低交替

### 语言文字检查
- [ ] 无 AI 痕迹（没有"此外""然而""值得注意的是""不禁""仿佛"等）
- [ ] 不用四字成语堆砌
- [ ] "的"字密度控制（一句话不超过两个）
- [ ] 动词优于形容词，用具体词替代抽象词
- [ ] 运用了白描/留白等中文技法

### 连贯性检查
- [ ] 与前文连贯，上一章的悬念有回应
- [ ] 伏笔有呼应或即将揭示
- [ ] 时间线一致，人物已知信息一致

## J8. 题材特定检查

根据小说题材类型补充以下检查项（与通用检查并用）：

**悬疑/犯罪类：**
- [ ] 有线索揭示（包括读者后来才发现的隐线索）
- [ ] 有新谜题提出（不能只解答不提问）
- [ ] 逻辑无漏洞，嫌疑人行为有合理动机
- [ ] 线索公平性：读者理论上可提前猜到真相

**言情/都市类：**
- [ ] 关系有进展（不能原地踏步五章）
- [ ] 有情感张力（暧昧、误会、心动至少占一种）
- [ ] 甜虐比例根据剧情阶段调整

**奇幻/科幻类：**
- [ ] 世界观前后一致，不推翻已建立规则
- [ ] 不为了方便临时加设定
- [ ] 力量体系有明确边界

**动作类：**
- [ ] 动作场面清晰，读者能想象空间位置
- [ ] 动作段落用短句短段，节奏快速
- [ ] 每场动作有不同亮点
